WriteEm

A Lightweight Alternative to Visual Studio.

About WriteEm

WriteEm is designed to provide a lightweight, streamlined environment to write and execute code, all while reducing the heavy resource usage often encountered with traditional IDEs like Visual Studio.

Write code in your preferred language and run it as a standalone app!

Key Features

  • [WIP] Supports multiple languages like C#, C++, WESC
  • [WIP] Custom script environments for each language
  • Runs with minimal overhead, requiring only the necessary runner
  • Open-source and community-driven

How to Use

  1. Download the WriteEm executable from the GitHub releases.
  2. Place your source file (e.g., main.cs, main.cpp) in the same directory as the executable.
  3. Run the executable, and it will automatically execute your code based on the language specified in the we_CheckEngine file.
